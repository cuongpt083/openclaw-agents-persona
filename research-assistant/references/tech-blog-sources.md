This file is now the overview for the China creator source registry.

## Use This Structure

1. Start with [tech-blog-sources-index.md](/home/cuongpt/Workspaces/openclaw-agents-persona/research-assistant/references/tech-blog-sources-index.md).
2. Open only the matching source profile under [sources/](/home/cuongpt/Workspaces/openclaw-agents-persona/research-assistant/references/sources).
3. If present, open the matching snapshot under [source-snapshots/](/home/cuongpt/Workspaces/openclaw-agents-persona/research-assistant/references/source-snapshots).
4. Only then open direct source pages or fallback lookups.

## Why This Exists

- reduce repeated token cost from loading the entire registry
- let the agent fetch only the sources relevant to the current topic
- support cheap rolling snapshots of recent posts and signals
